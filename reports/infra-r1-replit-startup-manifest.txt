Startup flow
  1. Replit opens workspace.
  2. User starts API Server / PlaySyncer Frontend / Project workflow.
  3. Workflow executes ./scripts/replit-bootstrap.sh.
  4. Bootstrap acquires exclusive lock via flock on .replit-bootstrap.lock.
  5. Installs dependencies only when node_modules is missing or workspace
     manifests/lockfile changed; stores hash in node_modules/.replit-bootstrap.hash.
  6. Builds shared TypeScript declarations with pnpm run typecheck:libs.
  7. Requires DATABASE_URL without printing it.
  8. Runs pnpm --filter @workspace/db run db:check.
  9. Runs pnpm --filter @workspace/db run db:migrate.
 10. Releases lock and exits.
 11. Workflow starts the dev server.
